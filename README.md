# AlbionPlayersRadar

Real-time Albion Online player detection Android app using VPN-based network capture.

## Features
- Real-time player detection (name, guild, faction)
- Color-coded threat radar (passive/faction/hostile)
- Zone-aware alerts (Black Zone hostile warnings)
- Protocol18 Photon deserialization
- VPN tunnel capture (no root required)

## Build
1. Upload all files to GitHub
2. GitHub Actions builds automatically
3. Download APK from Actions tab

## Tech Stack
- Kotlin + Android SDK 34
- Android VPNService (no root)
- Protocol18 Photon packet parsing
- WebView canvas radar overlay
